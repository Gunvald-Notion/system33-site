Enkel & Multippel Regresjon

The convergence formation. Walk the four Galdr steps. Each form iz a different reading of the same convergence.

First principle. A regression model iz not a surface in a cloud. It iz a convergence of forces. Each predictor Xⱼ iz an arm of force radiatin’ from the center at iz own angle — iz own axis of the space. With k predictors the actual map lives in k+1 dimensions; the diagram flattens those axes ontuh one plane so the eye can hold them, but the geometry iz preserved — each new variable opens another direction the bead can be pulled. The intercept b̂₀ iz the equilibrium scalar. Each arm contributes b̂ⱼ·xⱼ, an the bead migrates by tha vector sum. The POSITION shows where forces resolve geometrically; the NUMBER inside iz the scalar ŷ.

Formula Reference — Formelark Notation

Canonical formulas. b̂ not β̂, t_TS not t_obs, df = n − k − 1 fer multippel regresjon.

Enkel lineær regresjonKap 11.1
Yⱼ = b₀ + b₁·Xⱼ + εⱼ
One predictor, one arm.
Multippel regresjonKap 11.8
Yⱼ = b₀ + b₁·X₁ⱼ + b₂·X₂ⱼ + ... + bₖ·Xₖⱼ + εⱼ
k arms, k forces. Each coefficient holds when all others are held still.
Testverdi (generelt)Kap 9 + 11
t_TS = (θ̂ − θ₀) / se(θ̂)
Force in units of iz own uncertainty. df = n − k − 1.
R² & Justert R²Kap 11.3
R² = SSR/SST = 1 − SSE/SST
R²_just = 1 − (1−R²)(n−1)/(n−k−1)
Fraction of Y-variance the convergence catches.
F-statisticKap 11.5
F = MSR/MSE = (SSR/k) / (SSE/(n−k−1))
Whole-model test. H₀: all bⱼ = 0.
RSEKap 11.3
RSE = √(Σeᵢ² / (n−k−1))
Residual standardfeil — average gap.
VIFKap 11.8
VIFⱼ = 1 / (1 − R²ⱼ)
Multikollinearitet. VIF > 5–10 means arms pull in nearly the same direction.
KonfidensintervallKap 11.5
b̂ⱼ ± t_(α/2, n−k−1)·se(b̂ⱼ)
Range where the true bⱼ likely sits.
