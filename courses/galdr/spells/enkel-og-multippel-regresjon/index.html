Enkel & Multippel Regresjon

Energy converges at the center. Each predictor pulls from iz own direction with iz own color. The bead findz iz home where all the forces resolve.

First principle. A regression model is not a surface in a cloud. It is a convergence of forces. Each predictor Xⱼ iz an arm of force radiatin’ from the center at iz own angle — iz own axis of the space. With k predictors the actual map lives in k+1 dimensions; the diagram flattens those axes ontuh one plane so the eye can hold them, but the geometry iz preserved — each new variable opens another direction the bead can be pulled. The intercept b̂₀ iz the equilibrium scalar, where the bead sits when no predictor fires. Iz coefficient b̂ⱼ iz the strength of pull per unit of input along the arm. When predictors take values, each arm contributes b̂ⱼ·xⱼ, an the bead migrates by tha vector resolved across all dimensions. The final POSITION of the bead iz where the forces resolve geometrically; the NUMBER inside the bead iz the scalar ŷ. Two readouts braided.

Formula Reference — Formelark Notation

Canonical formulas. Use these symbols verbatim — b̂ not β̂, t_TS not t_obs, df = n − k − 1 for multippel regresjon.

Enkel lineær regresjonKap 11.1
Yⱼ = b₀ + b₁·Xⱼ + εⱼ
One predictor, one arm.
Multippel regresjonKap 11.8
Yⱼ = b₀ + b₁·X₁ⱼ + b₂·X₂ⱼ + ... + bₖ·Xₖⱼ + εⱼ
k arms, k forces. Each coefficient holds iz meaning when all others are held still.
Testverdi (generelt)Kap 9 + 11
t_TS = (utvalgsverdi − hypoteseverdi) / standardfeil
= (θ̂ − θ₀) / se(θ̂)
Force in units of iz own uncertainty. df = n − k − 1.
R² & Justert R²Kap 11.3
R² = SSR / SST = 1 − SSE / SST
R²_just = 1 − (1−R²)(n−1) / (n−k−1)
Fraction of Y-variance the convergence catches.
F-statistic (hele modellen)Kap 11.5
F = MSR / MSE = (SSR/k) / (SSE/(n−k−1))
df = (k, n−k−1)
Whole-model test. H₀: all bⱼ = 0. Reject when F > F_CV.
RSE — Residual standardfeilKap 11.3
RSE = √(Σeᵢ² / (n−k−1))
Average gap between data points and the convergence.
VIF — Variance Inflation FactorKap 11.8
VIFⱼ = 1 / (1 − R²ⱼ)
Multikollinearitet diagnostic. VIF > 5–10 means the arms pull in nearly the same direction.
Konfidensintervall for b̂ⱼKap 11.5
